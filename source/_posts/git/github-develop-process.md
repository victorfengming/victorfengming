### 合作完成开源项目流程
1. 首先需要在github上注册账号并登陆
2. 安装git，到github官网下载安装包。
3. 为github账号添加ssh key
4. fork你想参加的项目
5. clone项目到本地
6. 将项目原地址添加为远程仓库
7. 创建branch，用于添加自己的修改
8. 同步远程仓库
9. 将修改push到我们的github上
10. pull request
