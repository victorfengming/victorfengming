### 字符串相关
- 单引号不能定义多个字符串

- 字符串判断相等最好用`==`:
    (Scanner输入进的字符串不能这么判断)
    解释:`==`不仅仅比较值还比较内存地址

### 类型相关

- 数组不能嵌套
- 数组中的类型要一样,提前定义好
- 数组不能直接增加
- 函数的返回值类型要提前声明好

### OOP
- 类不能进行多继承
- 接口可以多实现,来补充了类的多继承

### 容器类数据

- List 集合就是python中的列表list

- Map 集合就是python中的字典dict

- Set 集合就是python中的集合set
